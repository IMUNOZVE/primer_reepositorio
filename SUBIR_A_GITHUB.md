# Cómo subir esta entrega a GitHub Desktop

1. Descomprime el ZIP de entrega.
2. Abre GitHub Desktop y selecciona tu repositorio `primer_repositorio`.
3. Pulsa `Repository > Show in Explorer`.
4. Copia dentro del repositorio todo el contenido de esta carpeta, respetando la estructura:
   - README.md
   - data/
   - dashboard/
   - docs/
   - images/
   - resources/
5. Vuelve a GitHub Desktop. Deben aparecer cambios en la pestaña `Changes`.
6. En `Summary`, escribe: `Entrega proyecto dashboard analisis datos`.
7. Pulsa `Commit to main`.
8. Pulsa `Push origin`.
9. Entra en GitHub web, abre tu repositorio y comprueba que se ven los archivos.
10. Copia el enlace público del repositorio y pégalo en la plataforma del curso en `Agregar un enlace`.

Importante: el repositorio debe estar público hasta que finalice la corrección.
